# Real Estate Management Module for Perfex CRM

A comprehensive real estate management module for Perfex CRM with customer management, real estate project management, plots management, plot booking system, accounts management system, and team management system.

## Features

- **Customer Management**: Integrated with Perfex CRM's client system for managing real estate customers
- **Real Estate Project Management**: Create and manage multiple real estate projects with detailed tracking
- **Plots Management**: Add and manage plots within projects with specifications, pricing, and availability
- **Plot Booking System**: Book plots for customers with payment tracking and staff assignments
- **Accounts Management**: Record transactions, track payments, and manage balances
- **Team Management**: Assign staff members to projects with role definitions

## Installation

1. Copy the `modules/realestate` folder to your Perfex CRM installation
2. Navigate to Setup > Modules in the admin panel
3. Install and activate the "Real Estate Management" module

## Module Structure

├── language/        # Language files
